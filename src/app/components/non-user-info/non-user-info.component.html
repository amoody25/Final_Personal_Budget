A personal budget building application!

Take control of your personal finances now!

What?

EzBudget is a personal finance application designed to help individuals manage their money! It allows users to create, track, and manage their own personal budgets while providing them with visual representations of their spending!

Why?

Personal finance is a huge undertaking and many don't even know where to begin. EzBudget was created to help people get a grip on their personal budget and help them improve their understanding of personal finance. There is a lack of personal finance web applications in the world that are simple and easy to use, so this aims to solve that problem.

How?

EzBudget uses Node.js and Angular along with Google's Firebase suite to aid users. Through rigorous testing and debugging this application was designed to be simple and fast to use. This application uses Angular as the framework to implement it as a single page application. Google Firestore is the primary method of data storage and retrieval. Google's Authentication services are also used to manage users of this application.
